# Goose's Kentucky National Guard Professions (KNGP)

A Project Zomboid mod that adds Kentucky National Guard–themed professions, items and cosmetics (including armbands and brassards) for use in builds compatible with B42+. This repository above all is for people to grab previous releases of the mod and see how it is being worked on

## What’s Included

- **Custom professions** representing Kentucky National Guard roles
- **Cosmetic items** (armbands/brassards) for player characters
- **Versioned folders** for compatibility with B42.13, B42.14, B42.15, and later (as supported)

---

## Installation

### Steam Workshop (recommended)
1. Subscribe to the mod on Workshop.
2. Enable the mod in Project Zomboid’s "Mods" screen.

### Manual install
1. Copy the `KNGP [B42]` folder into your Project Zomboid `Zomboid/Workshop/` directory.
2. Make sure you are not subscribed to the mod on the actual Workshop.
3. Enable the mod in Project Zomboid’s "Mods" menu.

---

## Compatibility

- Built for **Project Zomboid B42.13+**.
- Includes separate `42.13`, `42.14`, and `42.15` folders for backwards compatibility.

---

## Development / Mod Structure

- `KNGP [B42]/Contents/mods/Goose's Kentucky National Guard Professions [B42]/` is the main mod directory.
- Each version folder (e.g., `42.14/`) contains version specific `media/`, `lua/`, `scripts/`, and `textures/`.
